EaglerNetwork Server-Wide Rules

  1. No Hacking in Any Form Ban length depends on the severity of the incident

    Including but not limited to:

    1. Auto-clickers
    2. Xray packs
    3. Non-vanilla clients (EaglercraftX is fine)

    Exceptions can be made for this rule; ask an admin

  2. No Cheating in the Real World One-week ban

    This includes, but is not limited to:

    1. Sharing accounts
    2. Creating alt accounts
  3. Do Not Bring Conflict Outside the Server Warning or one-week ban (depends on severity)

    Do not start real fights over events within the server or cause real harm to people and their things.

  4. Do Not Cheat the Server Permanent ban (potentially less: depends on severity)

    This includes, but is not limited to:

    1. Ban evasion [PERMANENT ban]
    2. Assisting banned players [1 MONTH ban]
    3. Exploiting/degrading admins [1 MONTH ban]
  5. Admins Can Make Exceptions at Any Time

    This rule allows admins to ban people for any reason if the inciting event is deemed to be a "loophole" or clear taunt/jab.

    For minor offenses, a warning should be applied by the administrators before any punishment. For larger offenses, players should have recognised themselves and used common sense. An instant ban of appropriate length can be administered in this case.

This rule table is the baseline for server etiquitte. Assume, using common sense, additional rules for each of the servers. This means trying to find "loopholes" in the rules is potentially a bannable offense.
